Tinder Auto Like is a Chrome extension that runs entirely in the user's browser.

## Data Collection
This extension does not collect, transmit, sell, or share any personal or sensitive user data.

## Data Storage
All settings (such as age range, distance filters, and keywords) are stored locally using Chrome's storage API.
No data is sent to external servers.

## Third Parties
The extension does not communicate with any third-party services.

## Authentication
The extension does not handle authentication. Users must log in to Tinder manually.
